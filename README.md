# Smart Profile & Skill Manager 

This is a simple interactive Command-Line Interface (CLI) project built with Python. It demonstrates the practical use of core data structures like Dictionaries, Tuples, Lists, and Sets.

## Features:
* Add user profiles with fixed data (Birth Year, Blood Group) using **Tuples**.
* Manage multiple projects using **Lists**.
* Store unique skills and prevent duplicates automatically using **Sets**.
* Store everything in an organized way using a **Dictionary**.

## How to Run:
1. Clone this repository.
2. Run `python main.py` in your terminal.
